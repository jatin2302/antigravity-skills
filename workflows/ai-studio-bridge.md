# AI Studio Bridge Workflow

Use this workflow when importing a project built in AI Studio (80% completion) to Anti-Gravity for refinement and scaling.

// turbo-all
1. **Import**: Open the downloaded AI Studio zip folder in AG.
2. **Initialize**: Check for `package.json` or `index.html`.
3. **Launch**: Run `npm run dev` or open `index.html` in localhost.
4. **Audit**: Call the `ui-ux-pro-max` skill to run a design audit.
5. **Sync**: Push the initial import to GitHub to establish a baseline.
6. **Refine**: Apply "Human Polish" and advanced backend logic.
