# Logic-Flow MCP Skill

Bridges visual logic systems with Anti-Gravity's execution engine.

## Context
Use this skill when the user presents a logic flowchart, decision tree, or Mermaid diagram.

## Workflow
1. **Extraction**: Parse the logic flow from images or markdown diagrams.
2. **Translation**: Convert decision nodes into structured JSON or TypeScript logic layers.
3. **Integration**: Create endpoints or local MCP tools that execute this logic.
4. **Endpointing**: Provide the API endpoint and key for the logic system to be called by the frontend.

## Visual Standards
- Use Mermaid for all logic visualization within the chat.
- Prefer portable logic (MCP) over static file-based code when possible.
