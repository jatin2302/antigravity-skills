# Agency Builder

Build, scale, and optimize a digital marketing agency from scratch or improve an existing one.

## When to Use This Skill
- Starting a new SEO/marketing agency
- Creating service packages and pricing tiers
- Building SOPs for recurring deliverables
- Designing client onboarding workflows
- Setting up team structure and roles
- Creating operational dashboards
- Developing sales processes and pitch decks

## Framework

### 1. Service Architecture
- Define core services (SEO, PPC, Content, Web Dev)
- Create tiered packages (Starter, Growth, Enterprise)
- Set pricing based on deliverable complexity and market rates
- Document scope boundaries to prevent scope creep

### 2. Operations
- **Client Lifecycle**: Lead → Proposal → Contract → Onboarding → Delivery → Retention
- **SOPs**: Step-by-step procedures for every recurring task
- **Tools Stack**: CRM, PM, Reporting, Communication
- **Quality Gates**: Checklists before any deliverable ships

### 3. Team & Growth
- Define roles: Account Manager, SEO Specialist, Content Writer, Dev
- Create hiring templates and skill assessments
- Build capacity planning models
- Set KPIs per role

### 4. Financial Health
- Track MRR, churn, LTV, CAC
- Invoice automation and payment follow-ups
- Profitability per client and per service

## Usage
- "Help me create a 3-tier SEO service package with pricing."
- "Build an SOP for monthly SEO reporting."
- "Design a client onboarding workflow for my agency."
- "What tools should I use to run a 5-person agency?"
