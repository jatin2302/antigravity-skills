# API Orchestrator

Framework for resilient 3rd-party API integrations.

## Core Features
- **API Audit:** Deep research before integration.
- **Mocking:** Schema-first development.
- **Resilience:** Circuit breakers and backoff logic.
- **Secrets:** Strict `.env` enforcement.
