# Continuous Improvement Protocol

Systematic feedback loops to extract project knowledge into Knowledge Items (KIs) and update global rules.

## Core Loop
1. **Retro:** Log what worked/failed to `findings.md`.
2. **Extract:** Distill reusable patterns into `/knowledge/`.
3. **Invariants:** Update `GLOBAL_RULES.md` to prevent recurring errors.

## Install
```bash
ln -s ~/GitHub/antigravity-skills/skills/continuous-improvement-protocol ~/.gemini/antigravity/skills/
```
