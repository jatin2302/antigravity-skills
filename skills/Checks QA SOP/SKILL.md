# QA SOP Enforcer

Kill mistakes before clients see them. Validate every deliverable against scope and SEO best practices.

## When to Use This Skill
- Before delivering any SEO work to a client
- Auditing content for on-page SEO compliance
- Checking technical SEO implementations
- Validating monthly reports before sending

## QA Checklist

### Content QA
- [ ] Title tag present, unique, <60 chars, includes target keyword
- [ ] H1 present, matches page topic, only one per page
- [ ] Meta description present, <160 chars, includes CTA
- [ ] Target keyword in first 100 words
- [ ] Internal links: minimum 3 per page
- [ ] Images have descriptive alt tags
- [ ] No broken links
- [ ] No placeholder/Lorem ipsum text
- [ ] Grammar and spelling check passed

### Technical QA
- [ ] Page loads in <3 seconds
- [ ] Mobile responsive
- [ ] Schema markup implemented correctly
- [ ] Canonical tag present
- [ ] Robots meta: indexable (unless intentionally noindex)
- [ ] URL slug is clean and keyword-rich

### Scope Compliance
- [ ] All deliverables listed in scope are completed
- [ ] No out-of-scope work included without approval
- [ ] Deliverable format matches agreement (PDF, Sheets, etc.)
